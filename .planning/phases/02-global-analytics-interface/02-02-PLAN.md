# Plan: Branch Explorer & Switching

This plan implements the visual branch selection grid and the logic to teleport into campus-specific dashboards.

## Tasks

<task>
<read_first>
- `templates/super_admin/dashboard.html`
</read_first>
<action>
Implement the "Branch Explorer" section in the dashboard:
1. Create a grid of "Campus Passports".
2. Each passport shows: Flag, Campus Name, Status (Online/Offline), Quick Stats (Issues/Titles), and a "Switch to Branch" button.

</action>
